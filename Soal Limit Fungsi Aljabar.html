Limit Fungsi Aljabar Limit Kanan Kiri Limit Trigonometri

Limit Fungsi Aljabar

Soal 1 :

Hitung nilai limit berikut:

\[ \lim_{{x \to 1}} \left( \frac{2}{x^2 - 1} - \frac{1}{x - 1} \right) \]

Penyelesaian:

Pertama, kita sederhanakan ekspresi dengan memfaktorkan penyebut dari pecahan pertama: \[ x^2 - 1 = (x - 1)(x + 1) \] sehingga ekspresi menjadi: \[ \frac{2}{(x - 1)(x + 1)} - \frac{1}{x - 1} \]

Lalu, kita samakan penyebut kedua pecahan tersebut: \[ = \frac{2 - (x + 1)}{(x - 1)(x + 1)} = \frac{2 - x - 1}{(x - 1)(x + 1)} = \frac{1 - x}{(x - 1)(x + 1)} \]

Karena \(1 - x = -(x - 1)\), kita peroleh: \[ = \frac{-(x - 1)}{(x - 1)(x + 1)} = \frac{-1}{x + 1} \]

Sekarang kita substitusi \(x = 1\): \[ \lim_{{x \to 1}} \frac{-1}{x + 1} = \frac{-1}{1 + 1} = \frac{-1}{2} \]

Jadi, nilai dari limit tersebut adalah \(\frac{-1}{2} \).

Soal 2:

Hitung nilai limit berikut:

\[\lim_{{x \to 2}} \frac{{x^2 - 4}}{{x - 2}}\]

Penyelesaian:

Untuk menyelesaikan limit ini, kita bisa memfaktorkan pembilang:

\[\frac{{x^2 - 4}}{{x - 2}} = \frac{{(x + 2)(x - 2)}}{{x - 2}} = x + 2\]

Dengan demikian, \[\lim_{{x \to 2}} \frac{{x^2 - 4}}{{x - 2}} = 4\].

Jadi, nilai dari limit tersebut adalah \(4 \).

Soal 3:

Hitung nilai limit berikut:

\[ \lim_{{x \to 4}} \frac{\sqrt{x} - 2}{x - 4} \]

Penyelesaian:

Untuk menyelesaikan limit ini, kita dapat menggunakan trik rasionalisasi dengan mengalikan pembilang dan penyebut dengan konjugat dari pembilang:

\[ \frac{\sqrt{x} - 2}{x - 4} \cdot \frac{\sqrt{x} + 2}{\sqrt{x} + 2} = \frac{x - 4}{(x - 4)(\sqrt{x} + 2)} \]

Sederhanakan ekspresi:

\[ = \frac{1}{\sqrt{x} + 2} \]

Sekarang kita substitusi \( x = 4 \):

\[ \lim_{{x \to 4}} \frac{1}{\sqrt{x} + 2} = \frac{1}{2 + 2} = \frac{1}{4} \]

Jadi, nilai dari limit tersebut adalah \(\frac{1}{4} \).
