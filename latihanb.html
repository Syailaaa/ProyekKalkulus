Soal Limit

1. Apakah \( f(x) = \begin{cases} x + 2, & \text{jika } x < 1 \\ 3x - 1, & \text{jika } x \geq 1 \end{cases} \) memiliki limit di \( x = 1 \)?

Pembahasan: Limit kiri (\( x \to 1^- \)) adalah \( f(x) = x + 2 = 3 \), sedangkan limit kanan (\( x \to 1^+ \)) adalah \( f(x) = 3x - 1 = 2 \). Karena limit kiri tidak sama dengan limit kanan, maka limit tidak ada.

2. Apakah \( f(x) = \begin{cases} 2x, & \text{jika } x < 0 \\ -2x, & \text{jika } x \geq 0 \end{cases} \) memiliki limit di \( x = 0 \)?

Pembahasan: Limit kiri (\( x \to 0^- \)) adalah \( f(x) = 2x = 0 \), dan limit kanan (\( x \to 0^+ \)) adalah \( f(x) = -2x = 0 \). Karena limit kiri sama dengan limit kanan, maka limit ada dan nilainya adalah 0.

3. Apakah \( f(x) = \begin{cases} x^2, & \text{jika } x \leq 2 \\ 4, & \text{jika } x > 2 \end{cases} \) memiliki limit di \( x = 2 \)?

Pembahasan: Limit kiri (\( x \to 2^- \)) adalah \( f(x) = x^2 = 4 \), dan limit kanan (\( x \to 2^+ \)) adalah \( f(x) = 4 \). Karena limit kiri sama dengan limit kanan, maka limit ada dan nilainya adalah 4.
